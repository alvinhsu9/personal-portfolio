Overview

A mental health yoga studio that aims to help calm your mind. H큰x카 Yoga's website was designed to be functional mobile first. As a designer, I was challenged with making stylistic changes to the page as the user increased their screen size yet maintaining consistency with relation to the overall theme throughout the project.

The Final Product

Design Process

To start, I had to think about the layout and colours that I wanted to use for my page. Once I had my colours decided, I had to choose how to section off my website.


Since this was a single page website, I had to decide how to divide the page up in a way that allowed the user to flow through in a way that made sense.

I decided I wanted to have an animation for the burger when clicked, so, I had the burger become the logo on a click.

I then focused on how I wanted the page to change on different sized screens. I started by changing the layout of the banner image when a user first lands on the page by shifting around components.

Development

Throughout this project I used multiple JavaScript plugins. Once I implemented the JavaScript code for each plugin into my file, I proceeded to style them to be responsive, and to share the same colour scheme as the site.


Accordion

The first plugin I used is 'accordion'. This plugin was used to store information in a hidden panel, and if the user wanted to read more about that title, they could click to reveal the contents.

Gridder & Flickity

Following the 'accordion' plugin, I decided to include 'gridder'. The reason I chose to include this plugin is because of the built-in function of clicking an image and having it grow. I wanted to include this for my team section so that users could click into each individual team member to see what their role is within the company.


Here I turn the gridder into a-3 column grid when a user's screen size reaches laptop or monitor size for better spacing.


Next, I chose to include 'flickity'. This plugin allowed me to create a slideshow which I used for the events section.

Masonry

Finally, I decided to include the 'masonry' plugin, which allowed me to include my gallery section and have the images shift according to the screen size.

Calendar & Sign-up

Once I had finished coding up the single page, I decided to go back and include a signup and calendar page, which could be accessed from the site's banner buttons.


This section allowed for the user to see a weekly calendar layout. Users would be able to see different time slots on different days, for multiple months and years, including past ones.

Here, users could select multiple time slots that they would want to sign up for. As well, with the arrow icons around the month and year, they could navigate through to different weeks.


Next, I started on the sign-up form. I changed the parameters to make sense with a yoga studio member sign up.


This allows users to activate a drop-down menu for when they input their date of birth. It also includes check boxes and radio boxes for when they select their gender and goals.

Takeaways

Overall, this was a fun, yet challenging website to create. However, with this new knowledge of coding for mobile first, I have learned what is necessary to create fully responsive website. I also learned the power of JavaScript plugins and their effectiveness in completing a sites overall functionality. I hope to take the skills and knowledge obtained from this website and apply them, or even improve on them on my future projects.
