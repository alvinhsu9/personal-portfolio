Overview

Something that I had recently picked up during the COVID-19 Pandemic is photography. Collecting more visual memories for myself. I decided that I wanted to turn this into a blog-style website for myself to display all my photos for others to see. I decided to take the skills and knowledge that I had learned throughout my time at BCIT and apply it to the site.

The Final Product

Design

For the design portion of this project, I started off with the logo that I wanted to use. Since this was technically still under my personal brand, I decided to use the same logo as I had on my main website but change the text from designs to photography. I then shifted my attention to wireframing possible layouts that I could use. I knew since I had almost an identical logo across, that I should aim to try to have a different look to this website when compared to my design one.

For the landing page I wanted to have a full background image cover the page, as well as have the menu hidden behind banner text. When a user clicks the burger icon to open the menu, the banner text would fade away and reveal the navigation menu.

Next, I began to try out different layouts of my collections section. I decided for this page I wanted to display the collections as cards. This way users would see a sample image from each collection as well as a title so that they would be able to easily identify what that collection contains.

I then laid out my gallery page for each card. I wanted the user to immediately be greeted with all the images related to that collection. I decided to use a 3 row grid layout, as I felt it would be easiest for users to retain and see the content on the page.



Finally, I focused on what the modal would look like when a user clicked on individual images. I wanted this modal so that users could focus on individual images at a time that they were most interested in. I also wanted to be able to have a brief description or caption for each image so that the user would be able to understand where the image was taken.

Development

With my wireframes set up, I was ready convert it into code. I started by setting up the landing page. I ultimately decided to have the burger menu centered along with the rest of the content on the page as it made more sense hierarchically, rather than having it floating alone on the side.

Next, I set up my collections page. For this page I used isotope so that the cards would automatically re-arrange themselves when a user re-sizes or uses a different sized screen. This allowed for the cards to stay organized and easy to visualize. I placed the title of the card along the bottom below the image as I wanted users to see the image first, and then look to see what the title of the collection is.

Takeaways

Since I used the same logo and it was still connected to my personal brand, I felt this was a fun challenge to create a completely different layout of my pages. Making a blog for my photos has been something that I have been wanting to do for a while and I am very pleased with the final product of the website. I also have always wanted to make a super simplistic landing page like the one I made for this one. This was a very fun project to complete since it was my first time working with website focused on galleries and cards. I am very excited to continue to upload my photos and content onto this website in the future and hope to be able to continuously use and update it.
